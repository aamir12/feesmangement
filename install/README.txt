How to Install

1) First you need to install xampp or wamp server. But I recommend install xampp server.
2) Open your brower and type localhost/phpmyadmin and create database with name rainbow
3) After creating database, click on the database name rainbow which is available on left hand side of  database list
4) Click Import option and choose the file rainbow.sql which is available in install folder.

5) Your Software install Successfully. Now type in your browser localhost/rainbow
